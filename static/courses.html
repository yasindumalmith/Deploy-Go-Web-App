My Core Skills and Tools

Containerization and Security (Docker, Distroless)

  • **Docker Multi-Stage Builds:** Utilizing a separate build environment (`golang:1.22`) and a minimal runtime environment (`gcr.io/distroless/base`) for maximum efficiency and security.
  • **Artifact Management:** Pushing verified images to a container registry (e.g., Docker Hub, AWS ECR).

Orchestration and Reliability (Kubernetes)

  • **Declarative Deployment:** Using YAML manifests to define desired state, including **Deployment** (scaling/replicas), **Service** (load balancing), and **Ingress** (external access).
  • **Health Checks:** Implementing **Liveness** and **Readiness Probes** to ensure zero-downtime deployments and self-healing.

Automation and Quality (CI/CD, Go Testing)

  • **Continuous Integration (CI):** Implementing automated testing (`go test` using `main_test.go`) to prevent deployment of broken code.
  • **Go Backend:** Experience writing simple, robust web services in Go for cloud-native performance.
